# 飞书多维表格自动化插件 - 任务清单同步

## 使用方式

```bash
cd plugin

# 1. 安装依赖 & 编译
npm install
npm run build

# 2. 登录飞书开发者工具
npx opdev login -e feishu

# 3. 上传插件
npm run upload    # 按提示输入版本号（如 0.0.1）

# 4. 在开发者后台发布应用
```

上传后，回到[开发者后台](https://open.feishu.cn/app) → 多维表格自动化操作 → 配置图标和名称 → 版本管理与发布

## 自动化流程配置

1. 打开你的多维表格 → 机器人图标 → 创建自动化流程
2. 触发条件：**当记录被创建或更新时**
3. 操作：选择本插件（"任务清单同步"）
4. 在插件配置中映射字段：
   - **任务名称** ← 选择「任务名称」字段
   - **任务清单（所属清单）** ← 选择「任务清单」字段
   - **清单GUID映射** ← 填写 JSON（见下方）
5. 保存并启用
